# Suggested commands
- `npm test` : run Jest tests.
- `npm run build` : compile TypeScript to `dist/` and remove test artifacts from dist.
- `npm run lint` : run Biome checks.
- `npm run fmt` : run Biome formatter.
- `npm run fix` : apply Biome fixes.
- `npm run clean` : remove `dist/`.
- `openspec list --json` : inspect active OpenSpec changes.
- Useful Darwin shell commands: `ls`, `pwd`, `rg --files`, `rg <pattern>`, `sed -n 'start,endp' <file>`, `git status`, `git diff`.
